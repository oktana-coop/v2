# flow

This is an example project using automerge repo

## Development notes

### Recommended tooling/practices

* Package manager: npm
* Node version: v20.0.0 (as listed on `.nvmrc`)
* Commit style: [Conventional commits](https://www.conventionalcommits.org/)

### Install

```sh
$ npm install
```

### Run

```sh
$ npm run dev
```
